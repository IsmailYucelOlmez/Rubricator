# 🌍 Internationalization (i18n) System – Full Implementation (TR + EN)

## 🎯 Goal

Build a complete i18n system that:

* Supports Turkish (tr) and English (en)
* Allows runtime language switching
* Stores user language preference (Supabase + local cache)
* Can be changed from Profile Page
* Updates UI instantly without app restart

# ⚙️ LOCALIZATION SERVICE

Create service:

```dart id="localization_service"
class LocalizationService {
  static const _key = "app_language";

  Future<void> saveLocale(String code) async {
    final prefs = await SharedPreferences.getInstance();
    await prefs.setString(_key, code);
  }

  Future<String?> getSavedLocale() async {
    final prefs = await SharedPreferences.getInstance();
    return prefs.getString(_key);
  }
}
```

# 🔄 STATE MANAGEMENT (Riverpod)

---

## Locale Provider

```dart id="locale_provider"
final localeProvider = StateNotifierProvider<LocaleNotifier, Locale>(
  (ref) => LocaleNotifier(),
);

class LocaleNotifier extends StateNotifier<Locale> {
  LocaleNotifier() : super(const Locale('en')) {
    loadLocale();
  }

  Future<void> loadLocale() async {
    final saved = await LocalizationService().getSavedLocale();
    if (saved != null) {
      state = Locale(saved);
    }
  }

  Future<void> changeLocale(String code) async {
    state = Locale(code);
    await LocalizationService().saveLocale(code);

    // optional: sync with Supabase user profile
  }
}
```

# 👤 PROFILE PAGE – LANGUAGE SELECTOR

---

## UI Component

```dart id="language_selector"
class LanguageSelector extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final locale = ref.watch(localeProvider);

    return ListTile(
      title: Text("Language"),
      subtitle: Text(locale.languageCode == 'tr' ? "Türkçe" : "English"),
      onTap: () {
        showModalBottomSheet(
          context: context,
          builder: (_) => Column(
            children: [
              ListTile(
                title: Text("English"),
                onTap: () {
                  ref.read(localeProvider.notifier).changeLocale('en');
                  Navigator.pop(context);
                },
              ),
              ListTile(
                title: Text("Türkçe"),
                onTap: () {
                  ref.read(localeProvider.notifier).changeLocale('tr');
                  Navigator.pop(context);
                },
              ),
            ],
          ),
        );
      },
    );
  }
}
```

# ⚡ UX RULES

* language change must be instant
* no app restart
* persist across sessions
* fallback to English

---

# 🧪 EDGE CASES

* unsupported locale → fallback en
* null preference → default en
* slow storage → optimistic UI update

---

# 🚫 ANTI-PATTERNS

* ❌ hardcoded strings in UI
* ❌ rebuilding entire app manually
* ❌ storing locale only in memory

---

# 🎯 EXPECTED OUTPUT

User can:

* change language from profile page
* see app update instantly
* keep preference after restart
